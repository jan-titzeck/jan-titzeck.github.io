Schleifen

Schleifen sind praktisch, wenn man den gleichen Code öfters ausführen will, jedes Mal mit einem anderen Wert. (W3Schools)

Übung 1

Tipp - Verschachtelte Schleifen Schleifen können ineinander verschachtelt werden:
for (let x = 0; x < 0; x += 1) {
 for (let y = 0; y < 0; y += 1) {
   circle(x, y, 2);
 }
}

Du musst die Schleifen noch anpassen.

Übung 2

Tipp - Funktionen
  • translate(width / 2, height / 2) setzt den Koordinatenursprung auf die Mitte der Leinwand
  • rotate(0.087) rotiert das Koordinatensystem am Ursprung um 0.087 Radianten
  • rectMode(CENTER) setzt den Ursprung von Rechtecken auf ihren Mittelpunkt
Tipp - Modulo-Operator Der Modulo-Operator % berechnet den Rest der ganzzahligen Division. Mit der Bedingung i % 2 == 0 kann zwischen fill('black') und fill('white') gewechselt werden.
