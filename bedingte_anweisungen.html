Bedingte Anweisungen

Die if...else-Anweisung führt eine Anweisung aus, wenn eine angegebene Bedingung wahr ist. Wenn die Bedingung falsch ist, wird eine andere Anweisung in der optionalen else-Klausel ausgeführt. (MDN)

Übung 1

Tipp - if...else Füge diese if...else-Anweisung in die draw()-Funktion ein: if (/*Maus ist im Kreis*/) {
 // fülle den Kreis mit Schwarz
} else {
 // fülle den Kreis mit Weiß
}


Die Kommentare bei /*...*/ und // ... musst du mit Code ersetzen.
Tipp - Bedingung Seien die Variablen x und y die Entfernung der Maus zum Mittelpunkt und die Variable r der Radius des Kreises. So ist die Maus im Kreis genau dann, wenn x * x + y * y < r * r gilt (Satz des Pythagoras).
